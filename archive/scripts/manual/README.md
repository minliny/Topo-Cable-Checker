# Manual Testing Scripts

This directory contains manual execution scripts, legacy test files, and debugging utilities that were used during the early prototyping phases of the project.

**Important Note:**
- These files are **NOT** part of the formal automated testing suite.
- Formal automated tests (which run via `pytest`) are located in the `tests/` directory at the root of the repository.
- Scripts in this directory might be outdated, lack `assert` statements, and rely on `print` outputs for visual verification.
- They are preserved here purely for historical debugging or ad-hoc flow testing. Do not rely on them for CI/CD checks.
